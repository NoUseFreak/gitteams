# Github

## Permissions

`github.com > Settings > Developer settings > Personal access tokens`

 - `read:org`
 - `Projects - Read`
 - `Repositories - Read`

## Configuration

```
GITTEAMS_GITHUB_USERNAME=username    # Optional
GITTEAMS_GITHUB_TEAM=teamname        # Optional
GITTEAMS_GITHUB_TOKEN=yourtoken      # Required
```
